### Project overview

This project is a Progressive Web App catalogue platform built using HTML, CSS, JavaScript and a SQL database. The goal is to create a fast, installable web application that feels like a mobile app while running entirely in the browser. The application allows users to browse, search, filter and store items in a structured catalogue, create personal accounts, and contribute their own entries and reviews.

The project focuses on both front end and back end development. The interface is designed to be responsive and mobile friendly, while the server and database manage user accounts, saved items and submitted content. The app also demonstrates modern PWA features such as offline functionality, background syncing and installability.

### Main features:

User accounts with login and personal profiles
Search and filter system for browsing catalogue items
Individual item pages with detailed information
Ability for users to add items and submit reviews
Personal collections and favourites
Responsive design for mobile and desktop devices

### Progressive Web App features

Installable on mobile and desktop devices through a web manifest
Offline access using service worker caching
Background syncing when internet connection returns
App like navigation and performance
